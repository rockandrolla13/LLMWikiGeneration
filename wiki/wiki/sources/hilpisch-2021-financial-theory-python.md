# Financial Theory with Python: A Gentle Introduction

**Authors:** Yves Hilpisch  
**Year:** 2021  
**Type:** book  
**Markdown source:** `markdown_output/hilpisch-2021-financial-theory-python.md`

## Summary

Financial Theory with Python: A Gentle Introduction (2021) by Yves Hilpisch is a beginner-oriented textbook published by O'Reilly Media that teaches financial theory and Python programming simultaneously from scratch. It progresses through increasingly complex model economies — two-state, three-state, general static, and dynamic — to introduce core asset-pricing concepts (arbitrage pricing, martingale measures, risk-neutral pricing, CAPM, Black-Scholes-Merton) alongside executable Python code. The book is explicitly positioned as a prerequisite to Hilpisch's more advanced Python for Finance (2nd ed., 2018) and targets readers with only basic mathematics background (calculus, linear algebra, probability theory).

## Key Claims

- Finance and programming are now closely intertwined disciplines, with Python being one of the most widely used languages in the financial industry.
- Mathematical foundations, theoretical finance, and basic programming techniques are traditionally taught independently — this book integrates all three from the outset.
- Technological trends (online trading platforms, open-source software, open financial data, cheap cloud compute) have removed entry barriers to global financial markets.
- Finance today is a discipline of four closely interconnected types of languages: English, finance, mathematics, and programming ('A Four-Languages World').
- The book is explicitly a prerequisite/companion to Hilpisch's Python for Finance (2nd ed., 2018), filling the gap of ground-up instruction.
- A two-state static economy, despite its simplicity, is sufficient to introduce net present value, arbitrage pricing, martingale measures, market completeness, and mean-variance portfolios.
- The Black-Scholes-Merton and Merton Jump-Diffusion models are covered in the general static economy chapter (Chapter 5).
- Dynamic economies introduce stochastic processes; binomial option pricing and Monte Carlo simulation are demonstrated in Chapter 6.
- Expected utility maximization (representative agent framework) is the dominant paradigm for decision-making under uncertainty covered in Chapter 4.
