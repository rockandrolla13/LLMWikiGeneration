# Learning DAGs and Root Causes from Time-Series Data

**Authors:** Panagiotis Misiakos, [[entities/markus-puschel|Markus Püschel]]
**Institution:** ETH Zurich
**Year:** 2025 (arXiv: 2501.03130)

## Summary

This paper introduces DAG-TFRC, a novel method for learning directed acyclic graphs (DAGs) from time series under the assumption of few root causes. The method learns both instantaneous and time-lagged dependencies using a [[concepts/structural-vector-autoregression|structural vector autoregression]] (SVAR) model and discovers the location and timing of root causes.

## Key Contributions

### 1. Few Root Causes Assumption
- Time series generated by sparse input events at unknown nodes and time points
- Root causes are significant events that propagate through the graph
- Differs from prior work assuming i.i.d. noise inputs
- Extends the static SparseRC method to time series

### 2. DAG-TFRC Algorithm
- Learns window graph $\mathbf{W}$ containing instantaneous ($\mathbf{B}_0$) and lagged ($\mathbf{B}_1, ..., \mathbf{B}_k$) dependencies
- Uses continuous optimization with sparsity-promoting objectives
- Scales to thousands of nodes efficiently
- GPU-optimized tensor operations

### 3. Theoretical Results
- **Identifiability:** Proves window graph is uniquely determined from data
- **Global Minimizer:** Ground truth is unique minimizer of $L_0$ pseudonorm on root causes
- Based on non-Gaussian identifiability theory (LiNGAM)

## Technical Framework

### SVAR Model
$$\mathbf{x}_t = \mathbf{x}_t \mathbf{B}_0 + \mathbf{x}_{t-1} \mathbf{B}_1 + ... + \mathbf{x}_{t-k} \mathbf{B}_k + \mathbf{c}_t$$

where:
- $\mathbf{B}_0$: instantaneous dependencies (must be DAG)
- $\mathbf{B}_1, ..., \mathbf{B}_k$: lagged dependencies
- $\mathbf{c}_t$: root causes (assumed sparse)

### Optimization Problem
$$\min_{\mathbf{W}} \|\mathbf{C}\|_1 + \lambda_1 \|\mathbf{W}\|_1 + \lambda_2 h(\mathbf{B}_0)$$

where $h$ enforces acyclicity (NOTEARS-style constraint).

## Experimental Results

### Synthetic Data
- Superior performance over baselines (VAR-LiNGAM, DYNOTEARS, tsFCI, PCMCI)
- Scales to 4000 nodes with 36000 edges
- 1000x faster than VAR-LiNGAM for 1000 nodes

### Financial Data (S&P 500)
- Successfully clusters stocks by sectors
- Discovers major stock movements as root causes
- Validates few root causes assumption in real markets
- Root causes represent significant events (earnings, news, policy changes)

## Key Insights

1. **Stock Market Interpretation:** Root causes are rare, significant events affecting few stocks daily
2. **Instantaneous vs. Lagged:** Stock markets react near-instantaneously ($\mathbf{B}_0$ captures most influences)
3. **Sparsity:** The $L_1$ objective for root causes outperforms $L_2$ loss used in prior methods

## Baselines Compared

- VAR-LiNGAM, Directed VAR-LiNGAM
- DYNOTEARS, NTS-NOTEARS
- tsFCI, PCMCI
- TCDF (neural network method)
- SparseRC (adapted for time series)

## Applications

- Financial network analysis
- Causal discovery in econometrics
- Gene activation networks
- Climate and meteorological analysis
