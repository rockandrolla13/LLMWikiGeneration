# Epsilon-Operator Semantics for Counterfactuals

## Definition

Schulz's semantics using Hilbert's epsilon-operator: a counterfactual is true iff its consequent holds at an arbitrarily selected relevant antecedent-world, grounding counterfactuals' limited epistemic accessibility.
