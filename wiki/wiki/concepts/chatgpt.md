# ChatGPT

## Definition

OpenAI's conversational interface and gpt-3.5-turbo / gpt-4 family of chat-optimized models that accept a list of role-tagged messages (system, user, assistant) and return assistant replies, fine-tuned via RLHF for dialogue.
