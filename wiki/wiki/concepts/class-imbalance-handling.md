# Class Imbalance Handling

## Definition

Techniques to mitigate skewed label distributions in training data, including choosing imbalance-aware metrics (precision/recall/F1, PR curves), data-level resampling (SMOTE, Tomek links, two-phase learning), and algorithm-level adjustments such as class-balanced or focal loss.
