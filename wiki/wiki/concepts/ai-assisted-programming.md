# AI-Assisted Programming

## Definition

The practice of using generative AI tools (LLM-based code assistants, chat models, and IDE integrations) to support the full software development lifecycle — requirements, planning, coding, refactoring, debugging, testing, and deployment — rather than only autocomplete.
