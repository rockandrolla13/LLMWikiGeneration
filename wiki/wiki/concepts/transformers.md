# Transformers

## Definition

Neural network architecture based on [[self-attention]], without recurrence.

## Components

1. **Encoder** - Processes input
2. **Decoder** - Generates output
3. **Multi-Head Attention** - Parallel attention

## Impact

Foundation for BERT, GPT, and all modern LLMs.
