# Modular Systematic Trading Framework

## Definition

Carver's end-to-end, swappable pipeline: instruments -> per-rule forecasts -> combined forecast -> volatility target -> position sizing -> portfolio of subsystems -> cost/size tailoring, with independent, substitutable stages.
