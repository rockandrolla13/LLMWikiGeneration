# LangChain

## Definition

Open source framework providing composable abstractions (prompts, chat models, tools, output parsers, retrievers, runnables) for building LLM-powered applications in Python and JavaScript. First released by Harrison Chase in October 2022, it standardises integrations across LLM providers and packages common LLM patterns into reusable building blocks.
