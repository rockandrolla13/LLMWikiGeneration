# Few-Shot Learning

## Definition

An LLM's ability to perform a task after seeing only a handful of input/output examples embedded in the prompt, in contrast to zero-shot (no examples) and traditional supervised learning that requires large training sets; used in code generation to demonstrate desired patterns.
