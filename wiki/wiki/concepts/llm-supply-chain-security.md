# LLM Supply Chain Security

## Definition

The discipline of vetting the provenance and integrity of base models, fine-tuning datasets, model weights, plug-ins, and third-party APIs used in an LLM application. A compromised open-source model or a malicious dependency can act as a backdoor, embedding biases or vulnerabilities deep inside an otherwise secure application.
