# Xiaowen Dong

**Affiliation:** Department of Engineering Science, University of Oxford
**Role:** Departmental Lecturer

## Research Focus

Xiaowen Dong is a leading researcher at the intersection of [[concepts/graph-signal-processing|graph signal processing]], machine learning, and complex networks. His work focuses on utilizing graphs to model relational structure within data and developing novel techniques that bridge signal processing and network science.

## Background

- **PhD:** Swiss Federal Institute of Technology (EPFL), Lausanne, Switzerland (2014)
- **Postdoctoral:** MIT Media Lab
- **Current:** Departmental Lecturer, University of Oxford

## Key Contributions

### Graph Signal Processing for ML
- Co-authored the influential review "[[sources/dong-2020-gsp-for-ml|Graph Signal Processing for Machine Learning]]" (IEEE Signal Processing Magazine, 2020)
- Pioneered connections between GSP tools and machine learning architectures

### Graph Learning and Topology Inference
- Developed methods for learning graphs from smooth signals
- Work on "Learning Graphs from Data: A Signal Representation Perspective" (IEEE SPM, 2019)

### Gaussian Processes on Graphs
- Supervised [[entities/yin-cong-zhi|Yin-Cong Zhi]]'s doctoral work on [[sources/zhi-2024-gaussian-processes-graphs|Gaussian Processes on Graphs]]
- Contributions to spectral kernel learning and wavelet-based GPs

## Selected Publications

1. **Dong, X.**, Thanou, D., Toni, L., Bronstein, M., Frossard, P. (2020). "Graph Signal Processing for Machine Learning: A Review and New Perspectives." *IEEE Signal Processing Magazine*.

2. Zhi, Y.C., Ng, Y.C., **Dong, X.** (2022). "Gaussian Processes on Graphs via Spectral Kernel Learning." *IEEE Transactions on Signal and Information Processing over Networks*.

3. Opolka, F., Zhi, Y.C., Liò, P., **Dong, X.** (2022). "Adaptive Gaussian Processes on Graphs via Spectral Graph Wavelets." *AISTATS*.

4. **Dong, X.**, Thanou, D., Rabbat, M., Frossard, P. (2019). "Learning Graphs from Data: A Signal Representation Perspective." *IEEE Signal Processing Magazine*.
