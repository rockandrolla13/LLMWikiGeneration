# Markus Püschel

**Affiliation:** ETH Zurich
**Department:** Computer Science
**Role:** Professor

## Research Focus

Markus Püschel is a professor at ETH Zurich whose research spans signal processing, algorithm design, and causal discovery. His recent work includes developing efficient methods for DAG learning from time-series data.

## Research Areas

### Causal Discovery
- DAG learning from observational data
- [[concepts/structural-vector-autoregression|Structural vector autoregression]] models
- Few root causes assumption for time series
- Efficient algorithms for large-scale causal inference

### Signal Processing
- Fast algorithms and transforms
- Automatic program generation
- Performance optimization

### Algorithm Design
- Algebraic signal processing theory
- Linear algebra optimizations
- Sparse methods

## Key Publications

1. Misiakos, P., Püschel, M. (2025). "[[sources/misiakos-2025-dag-tfrc|Learning DAGs and Root Causes from Time-Series Data]]." *arXiv:2501.03130*.

2. Misiakos, P. et al. (2023). "SparseRC: Sparse DAG learning via few root causes." Earlier foundational work for DAG-TFRC.

## DAG-TFRC Contributions

The DAG-TFRC method co-developed with Panagiotis Misiakos:
- Learns window graphs from time-series data
- Assumes sparse root causes (significant events)
- Scales to thousands of nodes
- Provides identifiability guarantees
- Successfully applied to S&P 500 stock data

## Technical Innovations

### Few Root Causes Framework
- Time series generated by sparse input events
- Extends static SparseRC to temporal data
- L1-norm optimization for root cause sparsity
- GPU-optimized tensor operations

### Theoretical Results
- Identifiability proofs for window graphs
- Global minimizer characterization
- Connection to LiNGAM identifiability theory

## Applications

- Financial network analysis (stock market clustering)
- Gene activation network inference
- Econometric causal modeling
- Climate and meteorological analysis
