# Contradictions

Pages in this directory document conflicting claims between wiki sources.

Each contradiction page should:
- Name the two (or more) conflicting claims
- Cite the source pages where each claim appears
- State what the conflict is precisely
- Propose a resolution or mark as unresolved

Use the page_type: contradiction frontmatter field.
See [[concepts/]] and [[sources/]] for the pages involved.
