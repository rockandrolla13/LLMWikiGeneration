# LLM Wiki

An LLM-maintained personal knowledge base with Obsidian + OMEGA integration.

## What is LLM Wiki?

LLM Wiki is a system where you add source documents and the LLM maintains wiki pages, extracts entities and concepts, tracks contradictions, and synthesizes knowledge on demand.

## Quick Start

```bash
# Initialize a new wiki
/wiki-init

# Add a source document
/wiki-ingest path/to/document.md

# Ask questions
/wiki-query "What is the main contribution?"
```
